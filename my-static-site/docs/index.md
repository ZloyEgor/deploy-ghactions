## Технический стек

Проект реализован с использованием следующих технологий:

- **MkDocs** - генерация статического контента из Markdown
- **JavaScript/TypeScript** - интерактивные элементы пользовательского интерфейса
- **Bootstrap 5** - адаптивная система компоновки и стилей
- **TypeScript** - типизированная разработка JavaScript
- **SCSS** - расширенные возможности стилизации
- **Vite** - современный инструмент сборки фронтенда
- **GitHub Actions** - автоматизация процессов сборки и развертывания

## Архитектурные особенности

!!! info "Адаптивность"
    Сайт обеспечивает корректное отображение на устройствах различных размеров экрана.

!!! info "Темизация"
    Реализована поддержка светлой и темной тем с сохранением пользовательских предпочтений.

    Оптимизация включает минификацию ресурсов, сжатие файлов и эффективное кэширование.

## Функциональные возможности

Система обеспечивает следующие возможности:

1. **Автоматическое оглавление** - генерация навигации по разделам страницы
2. **Переключатель темы** - динамическое изменение цветовой схемы интерфейса
3. **Плавная прокрутка** - улучшенная навигация по странице

## Установка и запуск

```bash
# Клонирование репозитория
git clone https://github.com/ZloyEgor/deploy-ghactions.git

# Установка зависимостей Node.js
npm install

# Установка зависимостей Python
pip install -r requirements.txt

# Запуск в режиме разработки
npm run dev:mkdocs

# Сборка для продакшена
npm run build
```
